# Hardware Components Study Guide

This guide introduces common hardware components found in desktops, servers, routers and switches.

## Desktop
- Front panel: power switch, USB, audio
- Motherboard: CPU socket, RAM slots, PCIe slots, chipset
- Storage: SATA/NVMe drives

## Server
- Rack unit form-factor; hot-swap bays, redundant PSUs
- Network cards (1/10/25/40GbE) and management ports (IPMI)

## Network Gear
- Switch: many RJ45/SFP ports for L2 connectivity
- Router: WAN uplink, routing table, ACLs
