Goal: Goal: Configure a router-on-a-stick to support Data VLAN and Voice VLAN, create DHCP pools (including option 150) for phones, and verify phone provisioning.

Lab Steps

Step 1: Create VLANs and trunk

On switch create VLAN 10 and 20 and configure trunk to router interface.

Expected Commands:
  • vlan 10
    name DATA
  • vlan 20
    name VOICE
  • interface gigabitEthernet0/1
    switchport mode trunk

Step 2: Router subinterfaces

Create Gi0/0.10 and Gi0/0.20 with appropriate IPs.

Expected Commands:
  • interface gigabitEthernet0/0.10
    encapsulation dot1q 10
    ip address 10.0.10.1 255.255.255.0
  • interface gigabitEthernet0/0.20
    encapsulation dot1q 20
    ip address 10.0.20.1 255.255.255.0

Step 3: Create DHCP pools

Create DHCP pool for DATA and VOICE (option 150 for TFTP).

Expected Commands:
  • ip dhcp pool DATA
    network 10.0.10.0 255.255.255.0
    default-router 10.0.10.1
  • ip dhcp pool VOICE
    network 10.0.20.0 255.255.255.0
    default-router 10.0.20.1
    option 150 ip 10.0.0.10

Step 4: Verify phone DHCP

From a simulated phone, run: show dhcp lease or check assigned IP and TFTP server.

Expected Commands:
  • show ip dhcp binding
  • ping 10.0.0.10

Simulated Outputs

  • ping 10.0.0.10 ‚Üí reply
  • show ip dhcp binding ‚Üí binding: phone->10.0.20.45
