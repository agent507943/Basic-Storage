Lab Steps

Step 1: Configure SNMP community and host

Expected Commands:
  • snmp-server community MONRO ro 10
  • snmp-server host 192.0.2.5 version 2c MONRO

Step 2: Configure SNMPv3 user

Expected Commands:
  • snmp-server group SECLOCAL v3 priv
  • snmp-server user admin SECLOCAL v3 auth sha AuthPass priv aes 128 PrivPass

Step 3: Verify

Expected Commands:
  • show snmp user
  • show snmp group

  • show snmp group ‚Üí Group: SECLOCAL, v3, priv
