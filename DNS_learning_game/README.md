# DNS Learning Game

An interactive learning application for mastering DNS (Domain Name System) concepts, built using Python and Tkinter.

## Features

- **Study Mode**: Comprehensive DNS reference material
- **Quiz Game**: Interactive questions across three difficulty levels
- **Progress Tracking**: Score history and performance monitoring
- **Audio Feedback**: Different sounds for correct/incorrect answers
- **Visual Effects**: Confetti animations for engagement

## Getting Started

1. Ensure you have Python 3.x installed
2. Run the application:
   ```bash
   python app.py
   ```

## Game Structure

- **Easy Questions**: Basic DNS concepts and terminology
- **Medium Questions**: DNS record types and resolution process
- **Hard Questions**: Advanced security, performance, and troubleshooting

## Learning Objectives

Master essential DNS concepts including:
- DNS hierarchy and structure
- Record types (A, AAAA, CNAME, MX, NS, etc.)
- Resolution process and caching
- Security features (DNSSEC, DoT, DoH)
- Performance optimization
- Troubleshooting techniques

## Requirements

- Python 3.x
- tkinter (usually included with Python)
- Optional: pygame for enhanced audio (fallback available)

Built using the same framework as the OSI Model learning application.
